# Iris Flower Classification

This project is developed as part of the CodeAlpha Data Science Internship.

## Project Overview

The Iris Flower Classification project uses Machine Learning to classify iris flowers into three species:

* Setosa
* Versicolor
* Virginica

based on flower measurements such as:

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

## Technologies Used

* Python
* Pandas
* Seaborn
* Matplotlib
* Scikit-learn

## Machine Learning Model

* Random Forest Classifier

## Features

* Data Visualization using graphs and charts
* Machine Learning model training
* Accuracy evaluation
* Flower species prediction

## Output

* Pairplot graphs
* Accuracy score
* Classification report
* Predicted flower species
